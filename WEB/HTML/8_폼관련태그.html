폼관련태그

form태그는 html 웹문서에서 사용자가 입력하는 특정 양식의 데이터들을 제공하는 태그이다
form 태그 내의 input태그로 데이터를 받을 형태를 구성하여 해당 input의 값들을 서버로 전달한다.
* action : 작성한 데이터를 어떤 서버에게 전달할지에 대한 내용 기입
* method : 전달할 전송방식을 HTTP통신의 전송방식을 설정할 때 사용하는 속성
> get : 서버에 사용자가 입력한 값을 전달하면서 요청시 사용자가 입력한 값들이 url에 노출되는 방식 (전달값에 제한이 있음)
> post : 서버에 사용자가 입력한 값을 전달하면서 요청은 하되, url에 노출되지 않음(전달값을 볼 수 없고, 크기 제한 없음)

 

submit 버튼을 클릭하면 method 요청 전송방식으로 action에 지정한 경로로 사용자가 입력한 값 (key=value)을 전달한다

fieldset : 폼요소를 그룹으로 묶는 태그

legend : 묶은 폼요소에 명칭을 붙이는 태그
