제이쿼리 개요

jquery란?

존 레식에 의해 개발된 JS 경량화 라이브러리로,
복잡한 자바스크립트 구문들을 짧고 쉽게 개발할 수 있도록,
여러 기능들을 제공하는 라이브러리다.
jQuery의 장점
1.DOM과 관련된 스크립트들을 짧게 구현 가능하다.
2.일관된 이벤트 연결을 쉽게 구현 3.ajax(비동기통신),이벤트 처리 등을 폭넓게 지원
4.jQuery와 관련된 확장형 플러그인, 오픈 API를 지원하여 이를 통해 차트, 슬라이드 쇼, 테이블, 애니메이션 등을 구현 가능하다.

제이쿼리 연결

1. js 파일을 직접 다운로드 받아 연결하는 방법

2. CDN(Content Delevery Network)를 통한 연결

$(document).ready()

현재 웹문서를 로드한 후 ready()함수를 실행한다.
