속성 선택자

속성 선택자는 기본선택자 뒤에 붙여 사용한다.

요소[속성] : 특정 속성을 가지고 있는 객체 선택
요소[속성=속성값] : 속성 안의 값이 특정값을 단어로써 포함하는 객체 선택
ex) div[name='Hello'] : div 중 name 속성 값이 Hello인 요소를 선택
요소[속성명 ~= 속성값] : 해당 속성의 특정 단어를 단어로써 포함하는 객체 선택
요소[속성명 ^= 속성값] : 속성 안의 값이 특정 값으로 시작하는 객체 선택
요소[속성명 $= 속성값] : 속성 안의 값이 '속성값'으로 끝나는 객체 선택
요소[속성명 *= 속성값] : 속성 안의 값이 특정 값을 포함하는 객체 선택
